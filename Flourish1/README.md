# Description
This is a visual that was made with the data set that shows daily requests to the city on graffiti, which from this bar graph visual we see a upward trend over time. For the data I had to clean it since each data that was recorded had the year and hundreds of entries which I had to group into these categories. This data came from Stephen Barham which is a public data set that is updated daily.

## Axis values

X-Axis- Number of requests that were received each year to customer service to the city

Y-Axis- This groups all the years that were on the data to one to make easier to see
