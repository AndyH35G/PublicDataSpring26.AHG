# Description
This is a visual that was made with the data set that shows yearly requests to the city on graffiti removal, This data came from Stephen Barham which is a public data set that is updated daily. I decided to use the same dataset which I cleaned previously because this is visualization is able to show the benfits of this dataset. There is a random drop that occurs through out the year of 2025, there was steady rise that was occurring throughout the years recorded graffiti removal requests which the trend would keep ongoing but a drop occurred mid way through 2025. With 2026 just starting maybe the spike will most likely adjust to the trend we have been seeing throughout the years in the past. Additionally this data set will probably never be complete so the drop may just be because it is fol1owing the trend for the new year there isnt many reports at this point in time, which later on in the year there could be a steady rise that occurs in the data that will follow the trend of the previous years. From this the dataset can have limits but the benfits of knowing what is going on with the graffiti removal requests and show this to the city to show that there is a clear issue that is occuring with the amount of graffiti that is appearing on the streets.

## Axis values

X-Axis- This is the value of years which has the half way point of each year

Y-Axis- This is the number of reports of said year
